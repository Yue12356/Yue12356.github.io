Deepin(Linux下都适用)下VScode的升级

当visual studio code 弹出窗口可以升级的时候，我点开下载更新总要跳转网页去下.deb包。为了更加方便地更新vscode
（天呐，我已经懒得去打开网页了嘛），我想用命令行来操作

wget https://vscode-update.azurewebsites.net/latest/linux-deb-x64/stable -O /tmp/code_latest_amd64.deb

sudo dpkg -i /tmp/code_latest_amd64.dep

这样子就不用点开网页，不用等网页加载完再点击下载，不用等下载完成再打开到文件夹安装

我其实只需要打两行命令就好的事情（设置成快捷键或写成一个command就更方便了）（好吧，我写在这里就是因为我经常忘了中间的网址）
